SQLMemo

Bookmark and quickly insert useful queries with a context menu in SQL Developer 4

Project mantained by Stefano Cristalli


Description

This is an extension for Oracle SQL Developer 4, which allows the user to memorize useful queries and quickly insert them in the worksheet with only one click.
